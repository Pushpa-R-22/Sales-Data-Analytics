Sales Data Analytics System

Project Overview

This project is built using Python, MongoDB Atlas, Pandas, and Matplotlib to analyze sales data and generate business insights.

Features

- Store sales data in MongoDB Atlas
- Retrieve data using PyMongo
- Analyze sales performance using Pandas
- Visualize sales trends using Matplotlib
- Export reports to CSV format

Technologies Used

- Python
- MongoDB Atlas
- PyMongo
- Pandas
- Matplotlib

Files

- insert_data.py – Insert sales data into MongoDB
- read_data.py – Read sales data from MongoDB
- analysis.py – Perform sales analysis
- visualization.py – Generate charts
- export_csv.py – Export data to CSV
- sales_report.csv – Generated sales report

Key Insights

- Total Sales Analysis
- Average Sales Analysis
- Highest Selling Product
- Category-wise Sales Performance
